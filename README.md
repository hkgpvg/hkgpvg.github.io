# Hanyi's Poetry Works

A beautiful website showcasing the poetry of Hanyi, built with GitHub Pages.

## Features

- Elegant, literary design optimized for reading poetry
- Responsive layout for desktop and mobile devices
- Individual poem pages with beautiful typography
- Smooth scrolling navigation
- Featured poems on homepage

## Structure

- `index.html` - Homepage with featured poems and about section
- `style.css` - Custom styling with serif fonts and literary aesthetic
- `script.js` - Smooth scrolling and navigation interactivity
- `poems/` - Individual poem pages

## Adding New Poems

1. Create a new HTML file in the `poems/` directory (e.g., `poems/new-poem.html`)
2. Copy the template from an existing poem page
3. Update the content with your new poem
4. Add a new poem card to the homepage in `index.html`
5. Commit and push to see changes live

## Customization

- **Colors**: Edit CSS variables in `style.css` `:root` section
- **Fonts**: Currently using Cormorant Garamond and Lora from Google Fonts
- **Content**: Update text directly in HTML files
- **About Section**: Replace placeholder bio in `index.html`

## Deployment

This site is automatically deployed via GitHub Pages. Any push to the `main` branch will trigger a new deployment.
